PL/0 Parser/Code Generator

Authors - Daquaris Chadwick, Marcus Sooter

How to use the PL/0 Code Parser/Code Generator:

To compile, use the command `gcc main.c -o compile.out`

To run, use the command `./a.out <yourpl0file> [<options>]`

./compile –l –a –v         Print all types of output to the console
./compile –v               Print only the VM executiontrace to the console
./compile                  Print nothing to the console except for “in” and “out”
